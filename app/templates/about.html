Objetivo:

Api feito por um estudante python para demostrar o uso do Flask, Sqlalchemy e PostgreSQL

Os end-points podem responder com json file atraves da chave 'Content-Type'= 'application/json'

Gerais:

/ Pagina inicial ['GET']

/about Pagina sobre a api ['GET']

/create Prenche as tabelas ['GET']

Clientes:

/clientes Tabela com todos os clientes ['GET']

/clientes/add Pagina para adicionar um cliente ['GET', 'POST']

/clientes/id=cliente_id Pagina com apenas um cliente espeficico ['GET', 'POST']

/clientes/edit=cliente_id Pagina para editar um cliente ['GET', 'POST']

/clientes/delete=cliente_id Pagina para deletar um cliente ['POST']

/chamados Tabela com todos os chamados ['GET']
